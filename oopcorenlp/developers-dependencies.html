Dependencies

oopcorenlp depends on a great many external java libraries. These dependencies are specified in pom.xml and are managed by maven.

I'm waiting on the package maintainer to merge my PR, but until then you will need to build jVerbnet from my fork: https://github.com/rsadasiv/jverbnet/tree/bugfix/verbnet33. Other than that, all java dependencies are available from maven central: https://search.maven.org/. Please respect the licensing on any 3rd party libraries.

There are two non-java dependencies: wordnet and verbnet. These are available as http downloads and are conventionally installed in a directory named "data" which is a sibling to the oopcorenlp directory.

For automated build scripts, visit oopcorenlp_ci.

There are a few non-free external dependencies which require API keys. While oopcorenlp will work without them, the results will be better with them. Support the community and go sign up.
