Pipeline

This contains metadata about the configuration and execution of the oopcorenlp analysis. Included mostly for convenience and debugging purposes.

The structure of the PIPELINE json is as follows:

  • annotations
    • List of custom annotators and annotation descriptions
  • coreNlpProperties
    • List of CoreNLP pipeline properties
  • analysis
    • startTime
    • endTime
    • elapsedMS
    • host
